## 🚀 Features
- **Scroll-Driven Parallax**: Advanced scroll-linked animations mapped directly to user scroll position.
- **Cinematic Overlays**: Brutalist full-screen hamburger menu with staggered text reveals and double-sliding text effects.
- **Smooth Scrolling**: Implemented using Lenis for buttery smooth navigation.
- **Micro-Interactions**: Magnetic hover effects, scramble/glitch text decoding, and physics-based spring animations.
- **Responsive Architecture**: Fully responsive across mobile, tablet, and ultra-wide displays.

## 🛠 Tech Stack
- **Framework**: [Next.js (App Router)](https://nextjs.org/)
- **Core Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scroll**: [Lenis](https://lenis.studiofreight.com/)

## 💻 Local Development

1. Clone the repository
```bash
git clone https://github.com/prosenjitdey2143/prosenjit-dey-portfolio.git
cd prosenjit-dey-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
